# Paper Materials

This directory contains reviewer-facing material supporting the CEID colour-mixing study.

## Readable Manuscript Material

- [`author-note-replacements.md`](author-note-replacements.md): complete manuscript-ready replacement text.
- [`../pseudocode`](../pseudocode): pseudocode-only descriptions of CEID and the Unity digital-twin workflow.
- [`word`](word): separate editable Word document for each author note and one consolidated document.

## Supporting Data

- [`data/ceid_closed_loop_trace.csv`](data/ceid_closed_loop_trace.csv): all 24 primary CEID physical trials.
- [`data/target_spectrum.json`](data/target_spectrum.json): captured optimization target.
- [`data/strategy_comparison_summary.csv`](data/strategy_comparison_summary.csv): sanitized comparison summary and caveats.
- [`data/gravimetric_repeatability_measured.csv`](data/gravimetric_repeatability_measured.csv): experimentally validated raw readings and calculated statistics.
- [`data/gravimetric_repeatability_measured.png`](data/gravimetric_repeatability_measured.png): error-bar and coefficient-of-variation figure.
