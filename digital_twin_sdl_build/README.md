# Digital Twin SDL WebGL Build

This folder is the archived Unity WebGL export for the AISCIA digital-twin SDL reviewer demonstration.

Recommended access:

- Open the hosted build: https://souhilsid.github.io/Self-Driving-Lab-Digital-Twin-/

Local/offline use:

- Download this folder from GitHub.
- Serve it with a local web server instead of opening `index.html` directly.
- The original Unity export uses compressed `.gz` assets, so a local server may need gzip `Content-Encoding` configuration.

This is a binary reviewer build only. The Unity source project, CEID implementation source, hardware control source, LiveKit API secrets, and participant tokens are intentionally not included.
